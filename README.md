# Loans & Deposits Analysis (Power BI)

Анализ объемов кредитов и депозитов коммерческих банков по регионам.

 Проект включает:

Динамику кредитов и депозитов по годам

Региональное сравнение

Долю регионов в общем портфеле

Loans-to-Deposits Ratio (LDR)

Цель

Оценка структуры банковского сектора и выявление дисбалансов ликвидности между регионами.
